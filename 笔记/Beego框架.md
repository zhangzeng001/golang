## 安装

* 需要提前安装git

* 安装beego

  ```shell
  go get github.com/astaxie/beego
  ```

* 下载bee工具

  ```shell
  go get github.com/beego/bee
  ```

  

## bee工具使用

新建一个全新的Web项目

```shell
bee new ProjectName
```

创建开发api应用

```shell
bee api ProjectNames
```

运行项目

```shell
cd ProdectName;bee run
```

打包操作

```shell
bee pack
```

查看当前bee、Beego、Go版本

```shell
bee version
```
